Dispatcher Terminal

Creating a Job

To create a job in the dispatcher terminal first you must click on the driver for the job being created. Then, input the name of the client who the load was booked through. Next, input origin, destination, route length (as stated on the confirmation sheet), and delivery date. Once all this information has been inputted, click the submit button. You have now created a job.

Completing a Job

When a driver has completed delivery and confirmation has been received, then the job must be completed through on the application. To do this, simply scroll down on the dispatcher terminal page and go to the active jobs listings. Click on the job you wish to complete and click submit. You have now completed a job.

Pay Period

Reviewing and Adding Weekly Inputs

To review and add weekly inputs, please navigate to the Current Pay Period screen and select the Review & Add Weekly Inputs tab. Select the driver you wish to view or create inputs for from the Select Driver list.

Processing Payroll

To process payroll, please navigate to the Current Pay Period screen and select the Process Payroll tab. Select the driver you wish to process and carefully review the information. If there are any troubles, click the yellow Send Back to Edit button at the bottom of the page. If there are no troubles, click the green Send to Records button at the bottom of the page.

Manage Employees

Viewing and Editting Drivers

To view or edit drivers, navigate to the Manage Employees screen. The first section on the screen will display drivers. If you wish to change information of an existing driver, select the edit button at the bottom of that driver. Make the desired edits in the form, and then click the green update button or the close button to cancel. If you wish to add a new driver, select the Click here to add new drivers button, fill out the form, and select the green update button to submit of the close button to cancel.

Viewing and Editting Users

To view or edit users, navigate to the Manage Employees screen. The second section on the screen will display users. If you wish to change information of an existing user, select the edit button at the bottom of that user. Make the desired edits in the form, and then click the green update button or the close button to cancel.

Altering Registration Codes

To alter registration codes, navigate to the Manage Employees screen. At the bottom of the Manage Employees screen, the existing registration codes will be listed in the Dispatcher Code: and Admin Code: text fields. To alter a code, update the text to the desired code and select the blue update button.

View Records

Viewing Individual Driver Records

To view records for an individual driver, navigate to the View Records screen. In the Search by Driver or All Drivers panel, select the desired driver. The period for which you would like to see records for must be defined. To select the start date, either manually type the start date into the preformatted field, or click the calender icon for a helpful calender dropdown where you can select the appropriate date. Repeat the same process for the end date and click the Search button.

Viewing All Driver Records

To view records for all drivers, navigate to the View Records screen. In the Search by Driver or All Drivers panel, select the All Drivers option at the top of the list. The period for which you would like to see records for must be defined. To select the start date, either manually type the start date into the preformatted field, or click the calender icon for a helpful calender dropdown where you can select the appropriate date. Repeat the same process for the end date and click the Search button.
